# Contributing to PT Nusantara Digital

First off, thank you for considering contributing to PT Nusantara Digital!

## Branch Naming Convention
- `feature/` - for new features (e.g., `feature/contact-form`)
- `bugfix/` - for bug fixes (e.g., `bugfix/header-alignment`)
- `hotfix/` - for urgent production fixes
- `docs/` - for documentation updates

## Commit Messages
We follow conventional commits.
- `feat:` A new feature
- `fix:` A bug fix
- `docs:` Documentation only changes
- `style:` Changes that do not affect the meaning of the code
- `refactor:` A code change that neither fixes a bug nor adds a feature
